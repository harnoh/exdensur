# EXDENSUR 費用負担シミュレータ

GSK エキシデンサー（EXDENSUR / depemokimab）の医師向け面談支援ツール。
重症喘息／好酸球性副鼻腔炎の生物学的製剤について、患者の年間自己負担額と、
他剤からエキシデンサーへ切り替えた場合の軽減額（26週＝年2回投与による高額療養費の軽減）を可視化します。

**公開URL:** https://harnoh.github.io/exdensur/simulator

## 特徴
- 単一HTMLファイル・ビルド不要・**オフライン動作**（iPad現場運用想定）
- 2モード：他剤から切り替え（switch）／新規ナイーブ導入（naive）
- 開発仕様書 v1.0 準拠。受け入れ基準（§9）6シナリオを検証済み。

## 構成
- `simulator/index.html` — アプリ本体（`harnoh.github.io/exdensur/simulator` で配信）
- `index.html` — ルートから `/simulator/` へのリダイレクト

## 注記
算定はすべて当該薬剤のみの年間自己負担であり、他医療費の世帯合算・保険者独自の付加給付・
自治体助成・指定難病助成は考慮していません。実際の負担額は保険者の判断等により異なる場合があります。
